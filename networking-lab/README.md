# Networking Lab

Hands-on Networking fundamentals lab for **IT Support and Helpdesk** 

### Lab Overview
- **Tools Used**: Cisco Packet Tracer
- **Focus**: Practical networking skills required for entry-level IT jobs
- **Status**: In Progress
