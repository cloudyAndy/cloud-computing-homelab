# Cloud Computing Homelab

A hands-on cloud computing and IT infrastructure lab built to develop and demonstrate practical skills for **IT Support, Helpdesk, and Junior Cloud Administrator** roles.

### 🎯 Objective
To gain real-world experience with major cloud platforms, identity management, collaboration tools, automation, and troubleshooting scenarios.

### 🛠️ Lab Environments

| Platform              | Type                  | Status        | Key Skills Demonstrated |
|-----------------------|-----------------------|---------------|--------------------------|
| **Microsoft 365**     | E3 Trial              | **Completed** | User Mgmt, Exchange, Teams, SharePoint, Security |
| **Azure**             | Free Tier             | **In Progress** | VMs, Networking, Storage, RBAC, Resource Groups |
| AWS                   | Free Tier             | Pending       | EC2, IAM, S3, VPC |
| Others                | ...                   | ...           | ... |
