# Cloud Computing Homelab

A hands-on cloud computing and IT infrastructure lab built to develop and demonstrate practical skills for **IT Support, Helpdesk, and Cloud Administrator** roles.

### 🎯 Objective
To gain real-world experience with major cloud platforms, identity management, collaboration tools, automation, and troubleshooting scenarios commonly encountered in IT Support and junior cloud roles.

### 🛠️ Lab Environments

| Platform              | Type                  | Status     | Key Skills Demonstrated |
|-----------------------|-----------------------|------------|--------------------------|
| **Microsoft 365**     | E3 Trial / E5 Dev     | Active     | User admin, Exchange, Teams, Intune, Entra ID |
| **Azure**             | Free Tier + Student   | Active     | VMs, Networking, Storage, RBAC |
| AWS                   | Free Tier             | [pending]   | EC2, IAM, S3, VPC |
| Others                | ...                   | ...        | ... |
